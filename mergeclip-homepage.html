MergeClip

Efficient text file sharing with AI chat bot

What is MergeClip?

MergeClip is an intuitive macOS tool that efficiently merges multiple text files and copies them to your clipboard. It's designed to streamline your workflow with AI chatbots, making file sharing as easy as a bear hug.

Beary Good Features

  • Seamlessly merges multiple text files
  • Automatically excludes binary files from the picnic
  • Speaks many languages (based on file extensions)
  • Works as a Quick Action or from the command line den
  • Provides a friendly file count feedback in Finder

Easy Installation

Quick Action Setup

  1. Download MergeClip
  2. Unzip the package (it's like opening a jar of honey)
  3. Follow the bear-y simple installation guide

How to Use MergeClip

As a Quick Action

  1. Select your files or folders in Finder
  2. Right-click and choose "Quick Actions > MergeClip"
  3. Watch as MergeClip hugs your content onto the clipboard
  4. A friendly dialog tells you how many files got cuddles

Cozy Output Format

Filename1.ext
```ext
File1 content (wrapped in a warm embrace)
```

Filename2.py
```py
File2 content (snuggled with care)
```
